## About The Project

<div align="center">
    <img src="Images/Logo.jpg" alt="Logo" width="500">
</div>
</br>
</br>

* PBL is widely adopted at FPT University, helping students develop practical skills, critical thinking, and teamwork through real-world projects

* PBL implementation relies on multiple disconnected platforms for communication, meetings, file sharing, and collaboration, causing workflow interruptions.

* Lack of integrated academic features such as real-time collaboration, progress tracking, contribution evaluation, feedback, and peer review reduces PBL effectiveness.


Therefore:
* An all-in-one Webapp PBL system supporting the entire project lifecycle: project creation, team collaboration, and final evaluation.

* Integrated collaboration, communication, and assessment tools in a unified system to enhance transparency, efficiency, and learning outcomes.

* Five main actors: Admin, HeadDepartment, Staff, Lecturer, Student
